** Installation

*** Add the repository configuration
#+begin_src shell
echo "repository=https://github.com/c0pt4n/void-linux-zen/releases/download/latest" > /etc/xbps.d/10-zen.conf
#+end_src

*** Sync and install
#+begin_src shell
sudo xbps-install -S linux-zen6.19 linux-zen6.19-headers
#+end_src
